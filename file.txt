● It looks like you're in a bash shell on Windows, but trying to activate a
  Windows-style virtual environment. The path syntax is incompatible - in bash
  you need forward slashes and the correct path.

  Try this instead:

  ./venv/Scripts/Activate

  Or if you want to use the full path:

  source ./venv/Scripts/activate

  Note: On Windows with bash (Git Bash), the virtual environment scripts are in
  venv/Scripts/ (not venv/bin/ like on Linux/Mac), and you use forward slashes
  for the path.
